Listado de 102 Etiquetas HTML

Nombre de la Etiqueta Descripción Ejemplo de Uso Elemento Adicional
<a> Define un hipervínculo. <a href="https://www.example.com">Visita Ejemplo</a> Se puede agregar target="_blank" para abrir el enlace en una nueva pestaña.
<p> Define un párrafo de texto. <p>Este es un párrafo.</p> Puede contener otros elementos en línea como <a>, <strong>, <em>.
<img> Incrusta una imagen en la página. <img src="imagen.jpg" alt="Descripción de la imagen"> Es importante incluir el atributo alt para accesibilidad.
<div> Define una sección o contenedor en el documento. <div>Contenido del div</div> Usualmente usado para aplicar estilos o agrupar otros elementos.
<h1> - <h6> Define encabezados de diferentes niveles. <h1>Encabezado 1</h1> <h1> es el más importante, <h6> el menos importante.
